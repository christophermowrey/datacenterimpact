# Changelog

## 0.1.1 - Local interaction pass

- Added pointer-based map dragging with capture and release handling.
- Added wheel zoom scoped to the map viewport.
- Made map zoom controls functional and bounded.
- Added browser-location permission feedback without storing a location.
- Added local-only Docker binding on `127.0.0.1:3000`.
- Added the first AWS migration runbook and repeatable UI test checklist.

## 0.1.0 - Initial MVP

- Added the responsive Houston map-first demo interface.
- Added demo facility filters, result cards, overlays, and detail pages.
- Added transparent score ranges, confidence labels, Docker Compose, and README setup notes.
